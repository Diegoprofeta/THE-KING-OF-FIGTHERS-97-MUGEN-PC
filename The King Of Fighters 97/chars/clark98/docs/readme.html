Description

This archive contains OrochiKOF97's and Scal's MUGEN conversion of the character named Clark Steel.

The sprites of this Clark come from The King of Fighters 98, some of them have been edited to fit our vision of this character.

The gameplay and the system are mainly The King of Fighters 2002 oriented, no MAX mode though.

This conversion contains two modes that determines his moveset:
  • KOF02 mode based on his King of Fighters '98 moves
  • Retro mode based on his old moves.

Read the next section to learn how to configure those modes.

Configuration

The configuration file name is config.txt and is located in the root directory of the character.

This file allows you to configure the following points:

  • Game mode:
    Defines the game mode (KOF02 or Retro). That game mode can be toggled by pressing Start during a taunt if the functionality is enabled (see below).

    Values:
    • -1 = Selection key defines the mode:
      • X, Y or Z = KOF02 mode
      • A, B or C = Retro mode
    • 0 = KOF02 mode by default
    • 1 = Retro mode by default
  • Game mode toggle:
    Allows / disallows the player to toggle the game mode.

    Values:
    • 0 = Off, game mode cannot be changed during a match
    • 1 = On, game mode can be changed during a match
  • Sound set:
    Specifies the sound set to use.

    Values:
    • -1 = Depends on the game mode:
      • If game mode is KOF02, The King of Fighters 2002 sounds
      • If game mode is Retro, The King of Fighters '94/'98 sounds
    • 0 = Always The King of Fighters 2002 sounds
    • 1 = Always The King of Fighters '94/'98 sounds
  • Dodge:
    Allows / disallows the character to dodge like KOF Extra mode. If dodge is enabled, it can be perform by pressing A+B without holding forwards or backwards. If you hold one of the direction, a roll will be performed instead.

    Values:
    • 0 = Off, dodge is disabled
    • 1 = On, dodge is enabled
  • Power charge:
    Allows / disallows the character to power charge in order to increase its power bar level.

    Values:
    • 0 = Off, charge is disabled
    • 1 = On, charge is enabled
  • HSDM background:
    Selects the blue KOF02 or the green KOF02UM HSDM background effect.

    Values:
    • 0 = KOF02 HSDM background effect
    • 1 = KOF02UM HSDM background effect

Character's profile

This profile is taken from The King of Fighters Official Website.

At the beginning of development, the only differences in designs for Ralf and Clark was one had a hat and the other had a bandana, but Clark later got a pair of sunglasses to become the character he is today.

Clark basically looks a lot like Ralf, but since the designer in charge of adjusting this character worked on his hit accuracy, action speed and attack strength, their combat style is where the true differences between the two can be found.
