 Installation
+++++++++++++++
 DownLoad 1 file named "Chang_0101.zip" or "Chang_0731.zip".
 The "Chang_????.zip"  contains 21 files shown below.
  Chang.air		Chang7.act
  Chang.cmd		Chang8.act
  Chang.cns		Chang9.act
  Chang.def		Chang10.act
  Chang.sff		Chang11.act
  Chang1.act		Chang12.act
  Chang2.act		Chang.snd
  Chang3.act		ReadMe_J.txt (In Japanese)
  Chang4.act		ReadMe.txt (In English, this file)
  Chang5.act		Skill.zip (Containes commands)
  Chang6.act

 Unzip a file then, put these files into 1 folder named "Chang".
 Then Put "Chang" folder into your Mugen "charas" folder.

 Ex.)
   muk00101-----charas
              :    |____Chang
              :    :       |____Chang.air
              :    :       :
              :    :       :
              :    :       :

 Add this line in your "select.cfg" (or "select.def")in "data" folder.

 Chang, stages/******.def

 Replace ****** with your favorite background name.

 Ok, run your Mugen then the Chang's face is added your character
 select screen. Let's enjoy !!!

+++++++++++++++++++++++
 For Mugen 0627 users
+++++++++++++++++++++++
 Open "Chang.def". Then exchange like below.

	[Files]
	cmd     = Chang.cmd
	;cns     = Chang.cns      ;Constants
	;st      = Chang.cns      ;States
	cns    = Chang2.cns     ;Constants  For 0627
	st     = Chang2.cns     ;States For 0627
 And update it.

++++++++++++++++
 Commands
++++++++++++++++
 Unzip "Skill.zip".
 Please show "Skill_E.htm" in the "Skill" folder.
 Button layout is below...

  X Y Z
 (l ; :) : Deffault Keys
  A B C
 (, . /) : Deffault Keys

 X...Light Punch  ,  Y...Hard Punch  ,  Z...Recover
 A...Light Kick   ,  B...Hard Kick   .  C...Extra Hard Attack

++++++++++++++++++
 Featuers
++++++++++++++++++
 / Air recover or fall recover
   Press X+A or Z Button , when he can recover.

 / Quick forward or backward
   Press X+A or Z Button on the ground.

 / Guard cancel quick forward or backward (Costs half power gauge)
   Press X+A or Z Button , when he guarded.

 / Guard cancel Extra Attack (Costs half power gauge)
   Press Y+B or C Button , when he guarded.

 / He has 3 Level 3 super Arts made by me(M.M.R.).
